What is the purpose of JavaScript?

JavaScript is used to interact with content on websites as well as the user.

  • 1. JavaScript can modify HTML and CSS.
  • 2. JavaScript can react to user actions such as mouse clicks, pointer movements, and key presses.
  • 3. JavaScript can send information to servers or download/upload files.

What's the difference between how you reference a variable versus a string in a JavaScript statement?

When referencing a variable as a string quotes are required. Coders can use double quotes, single quotes, or backticks.

  • 1. Double quotes: "Hi World!"
  • 2. Single quotes: 'Hi World!'
  • 3. Backticks: `Hi World!`

What's the difference between a single value and an expression in a JavaScript statement?

A single value (literal value) is data that represents itself.
Single values can be numbers, strings, Booleans, null, and undefined.
An expression is a combination of one or more values, variables, operators, and function calls that produces a resulting value.
