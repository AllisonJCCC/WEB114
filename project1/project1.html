QUESTIONS

01. What is the purpose of JavaScript?

The purpose of JavaScript is to make web pages interactive. It is a scripting language that allows developers to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

JavaScript is used to add interactivity to web pages in a variety of ways, such as:

  • Validating form input
  • Creating dynamic menus and drop-downs
  • Adding animations and transitions

05. Why are external scripts preferable to scripts added directly within an HTML file?

External scripts are preferable because they can be reused across multiple HTML pages. This can save time and effort. External scripts also keep your HTML code clean and organized.
